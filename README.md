###  Power BI Dashboard –  Revenue-and-Profit-Analysis-of-Adventure-Works-Dataset Analysis

This repository contains a **Power BI (.pbix) dashboard** designed to analyze and visualize key business data for **AdventureWorks**.

####  Overview

The dashboard provides interactive insights into:

* Sales performance
* Customer behavior
* Product line analysis
* Revenue and profit trends

####  File Included

* ` Revenue-and-Profit-Analysis-of-Adventure-Works-Dataset.pbix` – Main Power BI report file

####  Features

* Dynamic and interactive visualizations
* Drill-down and filtering capabilities
* KPI indicators for quick decision-making
* Clean and user-friendly dashboard layout

####  Tools Used

* Microsoft Power BI
* Power Query (ETL operations)
* DAX (for calculated columns and measures)

####  Use Case

This dashboard is ideal for:

* Business analysts
* Data analysts
* Decision-makers looking for actionable insights

####  How to Use

1. Download the `.pbix` file
2. Open it using **Power BI Desktop**
3. Explore the report using filters and visuals
